## ⚙️ AI 이미지 생성 규칙 (절대 준수)

> **위반 시 이미지 사용 불가 — 배포 금지**

1. **모든 이미지는 1:1 정사각형 풀블리드(full-bleed)**
   - 이미지의 모든 픽셀이 콘텐츠로 채워져야 함
   - 레터박스(letterbox) / 필러박스(pillarbox) 절대 금지
   - 여백·배경 패딩·테두리 절대 금지

2. **"16:9 format", "wide", "banner", "wide format" 프롬프트 금지**
   - 직사각형 구도 키워드 사용 금지 → 정사각형 구도만 허용

3. **썸네일 = 대표이미지 통일**
   - 이미지 1개만 생성
   - `thumbnail_url` = `hero_image_url` (동일 URL 저장)
   - 상품 페이지는 `thumbnail_url` 우선 사용

4. **상품 페이지 대표이미지 컨테이너 규칙**
   - `.prod-image-wrap`: `padding-top: 100%` (1:1 정사각형, 모든 브라우저 호환)
   - 내부 img: `position:absolute; top:0; left:0; right:0; bottom:0; width:100%; height:100%; object-fit:cover`
   - lazy loading 금지 (지연 로딩 없음)

5. **이미지 프롬프트 필수 포함 요소**
   - `Full-bleed 1:1 square` + `no borders` + `no padding` + `no letterboxing` + `edge-to-edge`
