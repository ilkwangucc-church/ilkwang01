## ✅ 현재까지 완료된 작업

### 1. 교회 공개 홈페이지 (메인 화면)
- 메인 화면 구성 완료 (히어로 배너, 최신 설교, 예배 일정, 블로그 미리보기, 행사, 교회 소개, 연락처)
- 헤더(상단 메뉴바) 완성 — 스크롤 시 숨김/노출 처리, 번쩍임 제거
- 로고 + 보조 텍스트 세로 배치 완성

### 2. 교회 소개 페이지
- 담임목사 소개 페이지 완성
- 교회 비전 페이지 완성
- 교회 역사 페이지 완성
- 오시는 길(위치 안내) 페이지 완성
- 예배 안내 페이지 완성

### 3. 예배 / 설교 페이지
- 설교 목록 페이지 완성 (유튜브 연동 준비)
- 예배 안내 페이지 완성

### 4. 청년부 / 교육부 페이지
- 주일학교 페이지 완성
- 청소년부 페이지 완성
- 청년부 페이지 완성

### 5. 소식 / 공지 페이지
- 주보 페이지 완성 (PDF 뷰어 포함)
- 커뮤니티 페이지 완성
- 행사 안내 페이지 완성
- 갤러리 페이지 완성

### 6. 기타 공개 페이지
- 온라인 헌금 안내 페이지 완성
- 자료실 페이지 완성
- 문의(연락처) 페이지 완성 (문의 폼 포함)

### 7. 로그인 / 회원가입 화면 ⛔ 잠금 완료
- 로그인 화면 디자인 완성 — 흰색 + 회색 투톤 카드 스타일
- 로고 + 교회명 세로 배치
- 아이디 또는 이메일로 로그인 가능
- 비밀번호 보기/숨기기 버튼 적용
- 로그인 성공 시 성도 포털로 이동
- 회원가입 링크 및 비밀번호 찾기(이메일 방식) 링크 포함
- 버튼 → 하단 안내 텍스트 간격 10px로 통일
- ⛔ 이 화면은 완성 잠금 상태 — 어떤 수정도 금지

### 8. 회원가입 화면
- 회원가입 기본 화면 구성 완성

### 9. 성도 포털 대시보드 — 기본 구조 완성
- 사이드바 메뉴 구조 완성 (커뮤니티 그룹 + 관리자 그룹)
- 대시보드 홈 화면 완성 (환영 배너, 오늘의 말씀, 커뮤니티 빠른 메뉴, 성도의 하루 미리보기, 다가오는 행사)
- 사이드바 로고 배치 완성

### 10. 관리자 대시보드 메뉴 구성 완성
- 회원 관리, 교인 관리, 증명서 발급
